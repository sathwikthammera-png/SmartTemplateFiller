# Quick Start: AI Features Setup (3 Steps)

## 🚀 Get Started in 3 Minutes

### Step 1: Get API Key (1 min)
Go to https://console.anthropic.com → Sign up → Create API Key → Copy it

### Step 2: Set Environment Variable (1 min)

**Open PowerShell in the project folder and run:**
```powershell
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "your-api-key-here", "User")
```

**Then restart VS Code**

### Step 3: Run App (1 min)
```
python main.py
```

---

## ✨ Use AI Features

### Right Panel (Screen 2) has 3 Tabs:

**Tab 1: Legal Search**
- Type keyword → Click Search → Double-click result to insert

**Tab 2: Generate Clause**  
- Select placeholder → Add context → Click Generate → Auto-creates legal text

**Tab 3: AI Suggestions**
- Click button → Gets smart recommendations for your document

## 💡 Examples

### Creating an Affidavit
1. Load template with `{{affiant_name}}`, `{{facts}}`, `{{declaration}}`
2. Use "Legal Search" tab → Search "affidavit" → Insert relevant sections
3. Use "Generate Clause" tab → Select `{{declaration}}` → Generate professional text
4. View in middle panel → Export

### Creating a Sale Deed  
1. Load template with `{{seller}}`, `{{buyer}}`, `{{property}}`, `{{warranty}}`
2. Fill manually or use AI to generate professional clauses
3. Use "AI Suggestions" to add required legal sections
4. Export as PDF or DOCX
